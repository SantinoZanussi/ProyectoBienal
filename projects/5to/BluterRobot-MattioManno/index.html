Bluter Robot

Un robot pensado para aprender y enseñar programación de forma divertida.

5°1 Finalizado

Aplicación

Funcionalidades Principales

Inicio

Un menú principal que permite al usuario elegir entre las distintas funciones del robot, como programar o controlar manualmente.

Programacion

Una interfaz con bloques donde se pueden arrastrar y combinar acciones para que el robot las ejecute paso a paso.

Botonera

Un control manual con botones para mover el robot en diferentes direcciones y probar sus movimientos en tiempo real.

Objetivos del Proyecto

✓
Fomentar el pensamiento lógico y computacional a través de la interacción con un robot físico y una interfaz intuitiva.
✓
Crear una herramienta accesible, de bajo costo y fácil de replicar con materiales básicos y tecnología 3D.
✓
Promover el aprendizaje de programación y robótica en escuelas, talleres y hogares de manera divertida e interactiva.
✓
Despertar el interés por la tecnología de una forma accesible, amigable y divertida.

Tecnologías Utilizadas

Hardware

  • Arduino Nano – Módulo Bluetooth HC-05 – Driver Puente H – Motores DC – Servomotores – Estructura impresa en 3D

Software & Datos

  • Lenguaje Arduino C++ – Aplicación creada con Kodular – Diseño 3D realizado con Fusion 360
