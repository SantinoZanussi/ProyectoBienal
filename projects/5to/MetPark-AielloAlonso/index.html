MetPark

Estacionamiento automatizado con información en tiempo real

5°1 Finalizado

Aplicación Móvil

Funcionalidades Principales

Visualización en Tiempo Real

Consulta el estado de cada lugar del estacionamiento: libre, ocupado o reservado, actualizado instantáneamente.

Sistema de Reservas

Reserva tu lugar con antelación por tiempo determinado. Recibe un código de validación para acceder mediante teclado matricial.

Registro de Usuarios

Crea tu cuenta y accede a todas las funciones. Gestiona tus reservas y consulta tu historial de uso.

Objetivos del Proyecto

✓
Automatizar completamente el funcionamiento del estacionamiento para mejorar la experiencia del usuario
✓
Informar en tiempo real la disponibilidad de espacios mediante sensores infrarrojos
✓
Permitir a los usuarios reservar lugares con antelación a través de una app móvil conectada vía Wi-Fi
✓
Reducir el tráfico vehicular generado por la búsqueda de estacionamiento, disminuyendo las emisiones de CO₂

Beneficios del Sistema

✓
Reducción en la necesidad de personal para la gestión del estacionamiento
✓
Mayor comodidad para los usuarios con verificación de disponibilidad en tiempo real
✓
Ahorro de tiempo mediante el sistema de reservas anticipadas
✓
Reducción de emisiones de CO₂ (el tráfico por búsqueda genera 15-25% de gases invernadero)

Tecnologías Utilizadas

Hardware

  • Placa Arduino Mega
  • Módulo WiFi Wemos D1 Mini (ESP8266)
  • Sensores Infrarrojos FC-51
  • Servo Motor SG90
  • LEDs RGB NeoPixel
  • Pantalla LCD
  • Teclado Matricial

Software & Tecnologías

  • Programación Arduino (C++)
  • Desarrollo de App Móvil
  • Base de Datos
  • Comunicación WiFi
  • Diseño 3D (CAD)
  • Fritzing (Diseño de Circuitos)
