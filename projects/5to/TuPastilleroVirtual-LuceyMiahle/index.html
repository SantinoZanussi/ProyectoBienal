TuPastilleroVirtual

Soluciones inteligentes para no olvidar tu medicación.

5°1 En Proceso

Aplicación

Funcionalidades Principales

Alarma

TuPastilleroVirtual hará sonar una alarma cuando sea hora de tomar tus pastillas, ¡Así nunca olvidarás una sola dosis!

Administración manual

Puedes administrar las pastillas retirando la tapa y separando las pastillas por número

Objetivos del Proyecto

✓
Garantizar la comodidad de nuestros clientes.
✓
Agilizar el proceso de la ingesta de pastillas.
✓
Organizar efectivamente los medicamentos personales.
✓
Crear un recordatorio constante muy eficaz en alertar al usuario medicado.

Tecnologías Utilizadas

Hardware

  • Impresión 3D, servomotor, buzzer, interruptor, Arduino UNO, ESP32.

Software & Datos

  • C++, Firebase, Flutter, Shapr3D.

Danilo Lucey

Creador del sistema TuPastilleroVirtual, un proyecto que integra hardware y software para ofrecer una solución inteligente al manejo de medicamentos. Diseñé y programé tanto la aplicación móvil en Flutter como el sistema electrónico basado en Arduino UNO, que incluye un mecanismo circular de 8 compartimientos controlado por un servomotor, señales auditivas y visuales, y comunicación Bluetooth.
