Objetivos del Proyecto

✓
Obtener datos analógicos del potenciómetro mediante la micro:bit.
✓
Enviar los valores capturados a través de comunicación serial.
✓
Interpretar los datos recibidos para controlar la rotación de un vehículo virtual.
✓
Integrar hardware y software para simular un volante funcional en tiempo real.

Tecnologías Utilizadas

Hardware

  • Fueron utilizados dos Micro:Bit V2.2 y un potenciometro de 10K

Software & Datos

  • Se utilizo Unity Engine, framework .NET x4 y, lenguajes C# y Python
